🎯 Token Types

KAVACH supports 5 different honeypot token types

  • URL Tokens - HTTP/HTTPS URLs for web applications
  • API Key Tokens - Fake API credentials
  • Document Tokens - Traceable document identifiers
  • DNS Tokens - DNS records that log lookups
  • Email Tokens - Email-based honeypots

🔔 Alert Integrations

Get instant notifications when honeypots are triggered
