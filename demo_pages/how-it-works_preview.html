How It Works

From deployment to detection in 4 simple steps

🐳

$ docker run kavach:latest

Step 1: Deploy

Run KAVACH in Docker on any infrastructure. Takes 5 minutes. Zero complexity.

  • ✅ Works on any cloud (AWS, Azure, GCP)
  • ✅ Self-hosted or managed
  • ✅ No external dependencies
  • ✅ Single command to start

Step 2: Scatter

Generate honeypot tokens. Place them anywhere attackers look.

📦 Token Types:

  • • API Keys (sk_...)
  • • URLs (https://...)
  • • Documents (word.docx)
  • • DNS Records
  • • Email Credentials

🎯 Where to Place Them:

  • • Git repositories
  • • Email servers
  • • Slack channels
  • • Config files
  • • Shared drives
